# Maze Generator
Welcome to our maze generator game! In this game try to run through the maze as fast as you can without being pushed out by **the one**.

# Controls
## PC
- `w` - forward
- `a` - left
- `s` - back
- `d` - right
- mouse - look around with the camera
- `f` - toggle fog effect
- `` ` `` - toggle noclip
- `HOME` or `r` - respawn
- `,` - delete
- `n` - save
- `m` - load
- `q` - shoot ball
- `e` - toggle flashlight
- `p` - toggle music

### Pong Controls
- `w` - player 1 up
- `s` - player 1 down
- `i` - player 2 up
- `k` - player 2 down
